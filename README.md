# 이상운동증 진단 어플리케이션


이상운동증 진단 어플리케이션(이하 어플)은 신호 처리 기법을 활용하여 파킨슨병과 본태성진전 환자의 관리와 진단을 돕습니다.

자세한 내용 및 사용방법은 본문 자료(메뉴얼)를 참고해주시기 바랍니다.

## 1.어플리케이션 개요

이 어플은 기본적으로 환자의 떨림과 서동에 대한 객관적인 수치 기준을 제시하고 더불어 환자의 모니터링을 자동화하는 어플리케이션입니다. 이 어플을 통해 환자의 데이터를 수집할 수 있는 플랫폼을 제공하며 더 나아가 모호한 진단 기준을 개선하는 연구에 쓰일 것으로 기대되며, 현재 제작된 어플은 **다음의 특징**이 있습니다.

	환자 모니터링
	생체신호 분석 시스템
	환자 필터링 기능
	데이터 수집

이상운동증 진단 어플리케이션’은 실제 진단에 쓰이는 ‘UPDRS(Unified Parkinson’s Rating Scale)’, ‘CRTS(Clinical Rating Scale for Tremor)’, ‘나선그리기’, ‘직선그리기’, ‘글쓰기’기능을 지원합니다. > UPDRS와 CRTS는 설문지 형태로 이루어져있습니다. 전문의는 환자의 상태를 보며 전자 설문지를 작성합니다. 이때 CRTS의 세부항목으로 있는 그리기 수행은 5가지의 평가 수치를 제시하고 의사는 수치를 참고하여 진단을 내립니다.

## 3.기대효과
- 이상운동증이 현 시대에 심각한 질환 중 하나인 것을 알리고 초기발견이 중요한 질환인 만큼 미리 이상운동증에 대한 심각성을 인지할 수 있게 됩니다.

- 데이터로 증상의 경중과 생활 능력 상실 정도를 객관적으로 평가할 수 있게 되어 의사들의 주관적 판단에 의존하던 진단이 정량화된 데이터로 더 객관적인 진단이 가능하게 됩니다.

- 초기 환자 부터 말기 환자, 각 개인마다의 병의 진전 속도를 알 수 있게 될 것이고 치료 및 약 처방의 정도를 정량화된 데이터를 기반으로하여 더 체계적인 방법으로 환자의 데이터를 관리 할 수 있게되어 환자가 본인의 상태에 대해 인지하고 생활패턴을 바꾸는 등 능동적인 치료를 받을 수 있게 됩니다.

- 4차 산업혁명 기술 중 하나인 빅데이터를 본 연구와 함께 의료계에서 활용한다면 이상운동증에 대한 데이터를 축적하여 해당 분야의 연구소요 비용을 낮추고 보다 많은 의료진과 임상 연구원들이 치료와 연구 본연에 몰입할 수 있도록 도와줄 것으로 기대됩니다.
